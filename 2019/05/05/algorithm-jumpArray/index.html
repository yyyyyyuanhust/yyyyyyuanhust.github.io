数组跳跃游戏(2)

跳跃游戏1.0

给定一维数组 数组中每个元素代表从该位置可以向后跳跃的步数
问能否从第一个位置跳跃到最后一个位置
输入 一位数组arr
输出 能否从第一个位置到达最后一个位置
例如

分析

若可以到达第i个位置 则一定能到达弟i-1个位置
设置变量jump保存当前到达的位置
创建一个距离数组dis保存每个点可以到达的最远位置
扫描数组 不断更新最远位置
若到达某位置时 当前下标>最远位置 则无法达到
若扫描结束 则可以达到

跳跃游戏2.0

若给定的数组可以跳跃到最后位置求最小跳跃步数
输入 一维数组arr
输出 最小步数
例如

分析

使用jump保存结果

第一步应该往哪跳?
假设从第一个位置 第一步跳跃到dis[0]
那么从0到dis[0]之间的所有点都可能在第一次跳跃后达到
那么第一步应跳跃到这里面dis[i]中最大的位置
只有这样下一次跳跃才可以到更远的位置
用current变量保存假设跳跃到的店dis[0]
显然在到达dis[0]之前就应该跳跃
用pre变量保存应该跳跃的位置(dis[i]最大的位置)
扫描数组 当i超过了current肯定会有一次跳跃(在这之前已经发生) 此时变量jump+1
此时跳跃到的位置是pre
更新current为pre
同理 在到达pre之前 肯定发生了跳跃
直到扫描结束
