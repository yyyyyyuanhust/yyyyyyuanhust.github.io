移动端

移动端适配之rem布局

PC端/移动端区别

效果:
    PC:鼠标经过效果(伪类选择器)
    移动端:没有鼠标,只有轮播图(点击 滑动 效果)等
浏览器:
    PC:浏览器兼容(多内核)
    移动端:(webkit内核)手机适配(分辨率不同 显示页面效果不同)

视口

width=device-width 页面宽度等于设备宽度
user-scalable=no 不允许用户手动缩放页面
initial-scale=1.0 初始缩放比例1:1
maximum-scale=1.0 最大缩放比例1:1
minimum-scale=1.0 最小缩放比例1:1

Rem

rem = root element
rem获取html元素(根元素)的字体大小
chrome默认html的字体大小16px => 1rem=16px
1rem = 16px
? = 100px
不好计算
我们可以规定1rem = 100px
那么对于不同的分辨率(320,375,414…)
只需要使用媒体查询修改html{fontsize}即可

常见手机分辨率

手机 分辨率
iphone5/5s 320px
iphone6/6s 375px
iphone6sPlus 414px

以iphone5为基准

font-size:100px
那么
iphone6/6s
375:?=320:100
font-size=375x100/320=117.1875px;

iphone6P/6SP
414:?=320:100
font-size=414x100/320=129.375px;

以iphone6为基准

font-size:100px
那么
iphone5/5s
320:?=375:100
font-size=320x100/375=85.33px;

iphone6P/6SP
414:?=375:100
font-size=414x100/375=110.4px;

DPR

DPR:设备像素比

手机 分辨率 DPR
iphone5/5s 320px 2
iphone6/6s 375px 2
iphone6sPlus 414px 3

设计稿依照分辨率xdpr设计
一般分为640px(参照iphone5)和750px(参照iphone6)
使用时需要将font-size缩小才能正确显示

对于750px的设计稿
iphone6/6s font-size:50px;
iphone5/5s font-size:42.67px;
iphone6Plus font-size:55.2px
