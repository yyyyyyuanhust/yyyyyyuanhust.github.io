二、执行上下文

1、将代码分类可分为

全局代码
函数代码

2、全局执行上下文

在执行全局代码前将window确定为全局执行上下文
对全局数据进行预处理
    用var定义的全局变量===>undefined,添加为window的属性
    function声明的全局函数===>赋值,添加为window的方法
    this===>赋值为window
开始执行全局代码

3、函数执行上下文

在调用函数,准备执行函数体之前,创建对应函数的执行上下文对象
对局部数据进行预处理
    形参变量赋值实参==>添加为执行上下文的属性
    arguments===>赋值(实参列表),添加为执行上下文的属性
    var定义的局部变量--->undefined,添加为执行上下文的属性
    function声明的全局函数===>赋值,添加为执行上下文的方法
    this===>赋值(调用函数的对象)

三、作用域和作用域链

作用域

1、解释
    *一段代码块所在的区域
    *编写代码时就确定了,是静态的
2、分类
    *全局作用域
    *函数作用域
    *块作用域(es6)
3、作用:隔离变量,不同作用域下的同名变量不冲突

作用域与执行上下文

1、区别
    1.1
    *全局作用域之外,每个函数都会创建自己的作用域,作用域在函数定义时就已经确定了,而不是在函数调用时
    *全局执行上下文环境实在全局作用域确定之后,js代码执行之前创建
    *函数执行上下文是在调用函数时,函数体代码执行前创建的
    1.2
    *作用域是静态的,函数定义好就一直存在,不会变化
    *执行上下文是动态的,调用函数时创建,函数调用结束自动释放
2、联系
    *执行上下文环境从属于所在的作用域
    *全局上下文环境===>全局作用域
    *函数上下文环境===>对应的函数使用域

作用域链

1、解释
    *多个上下级关系作用域形成的链,它的方向是从下向上(从内到外)的
    *查找变量沿着作用域链查找
2、查找规则
    *当前作用域的执行上下文查找,找到返回,否则进入下一步
    *在上一级作用域中的执行上下文查找,找到返回,否则进入下一步
    *再次进行上一步的查找操作,直到全局作用域,未找到则抛出异常

排序算法

冒泡排序(bubbleSort)

bubbleSort:function
args:arr  目标排序数组
每一趟排序从左到右进行,相邻两个数比较后者大于前者进行交换
每一趟排序得到一个最大的数

例:排序[4,3,5,6,1,2]
第一趟   3 4 5 6 1 2    比较4和3 4>3 交换
        3 4 5 6 1 2    比较4和5 4<5 不交换
        3 4 5 6 1 2    比较5和6 5<6 不交换
        3 4 5 1 6 2    比较6和1 6>1 交换
        3 4 5 1 2 6    比较6和2 6>2 交换   
        此时第一趟结束 最大值6到达数组末尾
第二趟   3 4 5 1 2 6    比较3和4 3<4 不交换 
        3 4 5 1 2 6    比较4和5 4<5 不交换
        3 4 1 5 2 6    比较5和1 5>1 交换
        3 4 1 2 5 6    比较5和2 5>2 交换
        此时第二趟结束 第二大值5到达数组倒数第二位置
        ...
        ...
        ...

代码实现

function bubbleSort(arr){
    let len = arr.length;
    for(let i=0;i<len-1;i++){             //第一层的i仅作为控制当前数据长度的变量 在第二层中使用len-i减少长度
        for(let j=0;j<len-i-1;j++){
            if(arr[j]>arr[j+1]){
                let temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
            }
        }
    }
}

快速排序(quickSort)

quickSort:function 
args:[arr,l,r]  对数组arr的下标l-r范围的数据进行一次排序处理

思路:双指针递归 设指针i从做向右扫描,j从右向左扫描

例:排序[6,3,8,5,2,4,7,1]

6 3 8 5 4 2 7 1
i             j

首先选择一个基准 选择目标范围的第一个数作为基准 排序结束后基准左侧的所有数都小于等于基准,基准右侧的所有数都大于等于基准
这里基准arr[i]=6 令x=arr[i]

第一趟:j从右向左扫描 找比x小的值 此时arr[j]<x  (1<6)  进行赋值操作arr[i]=arr[j]
1 3 8 5 4 2 7 1
i             j

第二趟:i从左向右扫描,找比x大的值 
1 3 8 5 3 2 7 1
  i           j    3<6 继续向右扫描
    i         j    8>6 进行复制操作arr[j]=arr[i]

1 3 8 5 3 2 7 8
    i         j

第三趟:j从右向左扫描,找比x小的值
1 3 8 5 3 2 7 8
    i       j     7>6 继续向左扫描
    i     j       2<6 进行赋值操作arr[i]=arr[j]

1 3 2 5 3 2 7 8
    i     j

第四趟:i从左向右扫描,找比x大的值
1 3 2 5 3 2 7 8
    i     j
      i   j      5<6 继续向右扫描
        i j      3<6 继续向右扫描
          i      ij重合 扫描结束 进行复制操作arr[i]=x

此时的数组为
1 3 2 5 3 6 7 8
可以发现6左侧的数字都小于6 右侧的数字都大于6 
对两侧进行递归 即可完成快速排序,递归退出的条件 目标数组长度<=1

代码实现

function quickSort(arr,l,r){
    let i,j,x;
    if(l>=r){
        return; 
    }
    i=l;
    j=r;
    x=arr[i]
    while(i<j){
        while(arr[j]>x&&i<j){
            j--;
        }
        if(i<j){
            arr[i]=arr[j];
        }
        while(arr[i]<x&&i<j){
            i++;
        }
        if(i<j){
            arr[j]=arr[i];
        }
    }
    arr[i]=x;
    quickSort(arr,l,i-1);
    quickSort(arr,i+1,r);
}

堆排序(HeapSort)

实现过程:将要排序的数组写成完全二叉树形式
例如:将[2,1,4,3,5]进行堆排序

    2   [0]
   / \
  1   4    [1]   [2]
 / \
3   5        [3]     [4]

从左到右 从上到下 进行索引标记
对每个父结点进行heapify操作,使得以该父结点为根的子树构成大根堆

heapify

从父结点开始,值与左右子结点比较,将最大子结点和父结点交换,若父结点>左右子结点则不交换
如果发生了交换,对发生交换的子结点作为新的父结点再次进行heapify操作直到叶子结点
例1 对1进行heapify

  1     heapify     5
 / \    ======>    / \    (1和5进行了交换,此时的1作为父结点已经是叶子结点,不再继续heapify)    
3   5             3   1

例2 对2进行heapify

    2                         4
   / \      heapify          / \
  1   4     ======>         1   2   (2和4进行了交换,此时的2作为父结点已经是叶子结点,不再继续heapify)  
 / \                       / \
3   5                     3   5

例3 对2进行heapify

    2                         4                          4
   / \      heapify          / \        heapify         / \
  4   1     ======>         2   1       ======>        5   1   (2和4进行了交换,此时的2作为父结点不是叶子结点,继续对其heapify)
 / \                       / \                        / \
3   5                     3   5                      3   2

建堆

从后往前找第一个非叶父结点对其进行heapify操作,此操作向前一直进行,直到根结点,建堆操作结束后完全二叉树成为大根堆
所有的父结点都大于等于其左右子结点

堆排序步骤:

建堆、将根结点与最后一个结点交换,切断最后一个结点(数组的长度减1),这个结点为最大值,对剩余结点的根结点进行heapify操作,直到根只剩余结点

代码实现

/*
    堆排序(heapSort)
    swap:function
    args:arr,i,j  交换数组arr下标i和j数据

    heapify:function
    args:arr,n,i  对长度为n的数组arr的第i个结点进行操作,使arr[i]的左右子结点均不大于arr[i],此操作递归向下进行直叶子结点

    buildHeap:function
    args:arr,n    对长度为n的数组arr进行建堆操作,使所有父结点>=对应的子结点

    heapSort:function
    args:arr,n    对长度为n的数组arr进行堆排序

    例:排序[3,4,2,5,1]
        3          [0]            结点关系:父结点为i,左子结点:2i+1,右子结点:2i+2
       / \         / \            倒推:完全二叉树共有n个结点,则最后有一个结点index=n-1
      4   2      [1] [2]               最后一个lastParentNode = (index-1)/2
     / \         / \
    5   1      [3] [4]


    计算lastPNodes=Math.floor((5-1-1)/2)=1
    从arr[1]开始往前进行heapify操作 完成建堆
    对堆进行操作交换第一个结点和最后一个结点 交换一次长度减一 对剩余数据继续进行heapify,再交换......直到剩余数据只剩一个
    完成排序 
*/

function swap(arr,i,j) {
    let temp = arr[i];
    arr[i] = arr[j];
    arr[j] = temp;
}

function heapify(arr,n,i) {
    if(i>=n){
        return; 
    }
    let max = i;
    let c1 = 2*i+1;
    let c2 = 2*i+2;
    if(arr[c1]>arr[max]&&c1<n){
        max = c1;
    }
    if(arr[c2]>arr[max]&&c2<n){
        max = c2;
    }
    if(max!=i){
        swap(arr,i,max);
        heapify(arr,n,max);
    }
}

function buildHeap(arr,n) {
    let lastNodeIndex = n-1;
    let lastParentNode = Math.floor((lastNodeIndex-1)/2)
    for(let i = lastParentNode;i>=0;i--){
        heapify(arr,n,i)
    }
}

function heapSort(arr,n) {
    buildHeap(arr,n);
    for(let i=n-1;i>=0;i--){
        swap(arr,0,i);
        heapify(arr,i,0);
    }
}
//测试
let arr = [3,4,2,5,1];
heapSort(arr,5)
console.log(arr)//1,2,3,4,5
